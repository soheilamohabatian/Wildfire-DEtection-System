# Wildfire-DEtection-System
 An implementation of a K‑Nearest Neighbors (KNN) based system for wildfire detection from images.
 
The system is trained on a dataset containing images of both wildfire scenes and non-wildfire scenes, enabling it to classify a given image as either containing a fire or not.

System Overview:

The Wildfire Detection System utilizes a machine learning model, K-Nearest Neighbors (KNN), trained on a dataset, to distinguish between images depicting active wildfires and those that do not. This system can be used for various applications, such as:

- Early warning systems for firefighters and authorities.
- Automated monitoring of natural reserves or high‑risk areas.
- Image analysis tools for environmental research.

Dataset:
The system is trained on a dataset comprising:
- Images of wildfire incidents (e.g., burning forests, smoke plumes).
- Images of non-wildfire scenes (e.g., clear landscapes, normal vegetation, urban areas).
